Numify Auto-number your Zotero collections with hierarchical prefixes Controls how many levels deep the numeric prefix goes. For example, with depth 2 you get "1 Top" and "1.1 Sub" — but deeper sub-collections receive no prefix. Set to 20 (default 6) for unlimited practical depth. The character(s) placed between the numeric prefix and the collection name. Changing this immediately renames all collections in your library. Numify listens for collection events in Zotero and automatically: • Assigns a prefix when a collection is created • Renumbers siblings when a collection is deleted • Updates all affected prefixes when a collection is moved • Reorders siblings when you manually edit a prefix number Prefixes are stored directly in the collection name, so they sync across devices automatically. ⧉ GitHub — Source & Issues ☕ Buy me a coffee
